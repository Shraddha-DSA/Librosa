This repository documents my learning journey and experiments using Librosa for audio processing and analysis.

About

This repository contains:

- Basic audio loading and visualization
  
- Spectrograms and Mel-spectrograms
  
- Feature extraction (MFCC, Chroma, Spectral features)
  
- Audio transformations
  
- Small experiments and mini projects
  
- Notes and learning examples

The goal is to understand how audio signals work and how they can be used in Machine Learning, Speech Processing, and Audio Analysis.
